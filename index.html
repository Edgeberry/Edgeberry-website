Raspberry Pi for the IoT Edge

Adapting Raspberry Pi for the Challenges of IoT

Edgeberry is an open-source ecosystem that enhances the Raspberry Pi with hardware, software, and mechanical features for optimizing its usability as an IoT device. Aimed at fast and lean develop-and-deploy iterations, Edgeberry simplifies the process of bridging the physical world with the digital realm.

Edgeberry Device

Hardware Foundation for IoT

The Edgeberry device connects all components to the Raspberry Pi, providing it with a robust power supply, status indicators (LED and buzzer), and a unique identifier for cloud connectivity.

Together with the Enclosure that shields the system from real-world environmental challenges, Edgeberry transforms your Raspberry Pi into a rugged, production-ready IoT device. Designed with open tools like KiCad EDA and FreeCAD.

Hardware Cartridges

Custom Hardware for Each Application

Hardware Cartridges are the custom hardware component for each unique application, complementing the generic computer hardware. Add CAN bus, RS485, Console ports, analog I/O, and more to match your specific use case.

Design your own cartridges using the open-source KiCad EDA template, or explore existing designs. Create the perfect solution for your unique application - from automotive diagnostics to industrial automation.

Software

Edgeberry Device Hub

The Device Hub is the bridge connecting real-world devices with cloud applications. It's a self-hostable service that provides secure device management, telemetry collection, and bidirectional messaging.

Built with modern architecture featuring MQTT messaging, secure mTLS authentication, and WebSocket APIs. Deploy on-premises or in the cloud with Docker - you maintain full control of your data.

Integrate seamlessly with Node-RED, custom dashboards, and analytics platforms. The Edgeberry project is fully open-source with no features locked behind a paywall.
